Admin Centre

Functional Specifications & AI Transformation Opportunities

Project: Hyena Network Monitoring Platform
Interface: Admin Centre
Version: 1.0 (Prototype)
Date: November 2025

Executive summary

The Admin Centre is the platform-wide administration interface for Hyena platform operators. This master control panel enables organisation management, user administration, property oversight, system configuration, and platform-level analytics across all hotel chains using the Hyena platform.

Business value

By providing centralised control over multi-tenant white-labelled deployments, the Admin Centre enables Hyena platform operators to efficiently onboard new hotel chains, manage user access across organisations, maintain brand customisations, and monitor platform health—all from a single unified interface.

Core functionality

1. Organisation management

Complete CRUD operations for hotel chain organisations using the platform.

  • Organisation Registry: Master list of all hotel chains (Protea Hotels, City Lodge, etc.)
  • Branding Configuration: Upload organisation logos, set colour schemes, customise themes
  • Contact Management: Primary contacts, billing information, support escalation paths
  • Regional Settings: Timezone, language preferences, currency, localization (SA-specific)
  • Subscription Management: Plan tiers, property limits, feature access control
  • Property Portfolio View: All hotels within each organisation with health status

2. Property management

Configure and monitor individual hotel properties across all organisations.

  • Property Configuration: Hotel name, location, contact details, capacity metrics
  • Network Infrastructure: ISP details, bandwidth capacity, access point inventory
  • Staff Assignment: Link property managers, regional managers, technicians to properties
  • Performance Monitoring: Real-time incident volumes, resolution metrics, guest satisfaction
  • Multi-Tenant Isolation: Ensure data separation between competing hotel chains

3. User administration

Platform-wide user management with role-based access control.

  • Platform Users: Hyena super users, platform managers, technicians (cross-organisation)
  • Organisation Users: Property managers, regional managers (scoped to specific hotel chain)
  • Role Assignment: Super user, Hyena manager, Hyena user, technician, regional manager, property manager
  • Access Control: Define which users can access which organisations and properties
  • Activity Auditing: Track user logins, actions, data access for security compliance
  • Bulk Operations: Mass user creation for new organisation onboarding

4. Platform-level incident management

Cross-organisation incident visibility and emergency intervention capabilities.

  • All-Organisation View: Every active incident across entire platform
  • Organisation Filtering: Filter to specific hotel chain or property
  • System-Wide Patterns: Identify issues affecting multiple organisations
  • Emergency Intervention: Hyena staff can assign/reassign incidents across organisation boundaries
  • Escalation Management: Handle complex issues requiring Hyena platform expertise

5. Analytics & reporting

Platform-wide performance metrics and business intelligence for strategic planning.

  • Platform Health Metrics: Total active incidents, average resolution times, customer satisfaction across all organisations
  • Organisation Comparison: Benchmark performance between different hotel chains
  • Growth Metrics: User adoption rates, incident volumes over time, feature utilisation
  • SLA Compliance: Track service level agreement adherence for each organisation
  • Revenue Analytics: Organisation subscription values, growth trends, churn risk indicators

AI transformation opportunities

AI can transform the Admin Centre from a manual configuration and monitoring tool into an intelligent platform operations assistant that automates onboarding, predicts scaling needs, and provides strategic business intelligence.

1. Automated organisation onboarding

Current State: Manual setup of new hotel chains with configuration wizards.

AI-Enhanced Future: Intelligent onboarding assistant that learns from past deployments and automates setup.

  • Smart Configuration: AI suggests optimal settings based on organisation size, region, property types
  • Automated User Provisioning: Creates user accounts and assigns roles based on organisational structure
  • Branding Auto-Detection: Extracts colours and styles from uploaded logo to auto-generate theme
  • Property Import Intelligence: Parses provided property lists to auto-create hotel records
  • Network Discovery: Automatically detects and catalogs network infrastructure at new properties
  • Best Practice Application: Applies lessons learned from successful past deployments

2. Predictive capacity planning

Current State: Reactive response when organisations outgrow current infrastructure or subscription limits.

AI-Enhanced Future: Proactive forecasting of infrastructure and scaling needs.

  • Growth Forecasting: Predicts when organisations will need additional properties, users, or features
  • Infrastructure Sizing: Recommends optimal infrastructure based on expected incident volumes
  • Resource Allocation: Suggests technician staffing levels for new property openings
  • Upsell Opportunities: Identifies organisations ready for premium features or additional properties
  • Churn Risk Detection: Flags organisations with declining engagement or satisfaction for proactive retention

3. Intelligent anomaly detection

Current State: Admin reviews metrics manually to identify unusual patterns.

AI-Enhanced Future: Automated detection of platform-wide anomalies and potential issues.

  • System Health Monitoring: Detects unusual incident patterns that may indicate platform bugs
  • Security Threat Detection: Identifies suspicious user activity, unauthorized access attempts, data breaches
  • Performance Degradation Alerts: Warns when response times or resolution rates deviate from normal
  • Multi-Organisation Correlation: Identifies when issues affect multiple hotel chains simultaneously
  • Automated Escalation: Routes critical platform issues to appropriate Hyena engineers

4. Ai-powered business intelligence

Current State: Static reports showing historical platform metrics.

AI-Enhanced Future: Conversational BI assistant providing natural language insights and predictions.

  • Natural Language Queries: "Which organisations have highest incident rates this month?" → Instant ranked list
  • Predictive Analytics: "What will Q4 incident volume look like?" → Forecast with confidence intervals
  • Root Cause Analysis: "Why is Organisation X underperforming?" → Multi-factor analysis
  • Competitive Intelligence: "How do we compare to industry benchmarks?" → Market positioning insights
  • Automated Reporting: Generate executive summaries, board presentations, investor updates on demand

5. Smart user management & access control

Current State: Manual role assignment and access provisioning.

AI-Enhanced Future: Intelligent role suggestions and automated access lifecycle management.

  • Role Recommendation: AI suggests appropriate roles based on job title, organisational position, responsibilities
  • Access Anomaly Detection: Flags unusual access patterns (e.g., user accessing competing organisations)
  • Automated Deprovisioning: Detects inactive users and suggests account deactivation
  • Compliance Monitoring: Ensures POPIA compliance in data access and user permissions
  • Privilege Escalation Detection: Alerts when users gain inappropriate access levels

6. Platform optimisation recommendations

Current State: Platform improvements based on manual observation and user feedback.

AI-Enhanced Future: Data-driven optimisation suggestions for platform enhancement.

  • Feature Utilisation Analysis: Identifies underutilized features that could be promoted or deprecated
  • UX Improvement Suggestions: Detects workflow patterns indicating usability issues
  • Performance Optimisation: Recommends database optimisations, caching strategies based on usage patterns
  • Custom Feature Opportunities: Identifies unique needs that could be built into platform features
  • Integration Recommendations: Suggests third-party integrations based on organisation requests

Conclusion

The Admin Centre provides Hyena platform operators with comprehensive tools to manage multi-tenant white-labelled deployments at scale. By centralising organisation management, user administration, and platform oversight, the Admin Centre enables efficient operation of a SaaS platform serving multiple competing hotel chains while maintaining strict data isolation and brand customisation.

AI integration transforms the Admin Centre from a manual administration tool into an intelligent platform operations assistant that automates routine tasks, predicts future needs, and provides strategic business intelligence—enabling Hyena to scale efficiently while maintaining exceptional service quality across all customer organisations.
