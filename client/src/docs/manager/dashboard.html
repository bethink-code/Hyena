Manager Dashboard - Functional Specification

Overview

The Manager Dashboard provides property managers with a comprehensive view of network incidents and property health across their assigned properties. Managers can monitor active issues, track resolution progress, and create new incident reports.

Key Features

1. Summary Metrics

  • Active Incidents: Total count of all non-resolved incidents (new, assigned, in_progress, on_hold)
  • Critical Priority: Count of critical-priority incidents requiring immediate attention
  • In Progress: Incidents currently being worked on by technicians
  • Completed Today: Incidents resolved during the current day

Clicking any metric card navigates to the filtered incident list page.

2. Property Status Cards

Visual cards showing real-time status for each assigned property:

  • Healthy: No active incidents (green indicator)
  • Degraded: 1-2 active incidents (yellow indicator)
  • Critical: 3+ incidents or any critical-priority incident (red indicator)
  • Offline: Property experiencing connectivity loss (gray indicator)

Clicking a property card filters the incident list to that property.

3. Report Incident

Blue "Report Incident" button in top-right corner allows managers to create new incidents with:

  • Title and description
  • Category (Network, Load Shedding, ISP/Fibre, Weather, Guest Experience, Maintenance)
  • Priority level (Low, Medium, High, Critical)
  • Property assignment
  • Optional technician assignment
  • Location details
  • Event type (Reactive, Proactive, Environmental)
  • Metadata for SA-specific incidents (load shedding stage, ISP provider, weather conditions)

Navigation Pattern

The dashboard follows the Universal Navigation Pattern:

  1. View Summary: Dashboard shows clickable metric cards and property cards
  2. Filter by Metric: Click metric card → Navigate to /manager/incidents?status=...
  3. Filter by Property: Click property card → Navigate to /manager/incidents?propertyId=...
  4. View Details: Click incident in list → Detail panel slides in from right

User Roles & Permissions

  • Managers can view incidents for their assigned properties only (typically 3 properties)
  • Managers can create, edit, and take actions on incidents
  • Managers can assign/reassign incidents to technicians
  • Managers cannot access system configuration or user management

Data Refresh

Dashboard data automatically refreshes when:

  • New incidents are created
  • Incident status changes
  • Technicians update incident progress
  • User navigates back to dashboard from other pages
