Admin Center - Portfolio Dashboard

Overview

The Portfolio Dashboard provides system administrators with a comprehensive view of all properties, users, and platform-wide metrics. Admins have full access to configuration, user management, and cross-property analytics.

Key Features

1. Portfolio Overview KPIs

Top-level metrics displayed in card widgets:

  • Total Properties: Count of all properties in the portfolio
  • Active Incidents: Total active incidents across all properties (with percentage change)
  • Guest Satisfaction: Average satisfaction score (4.2/5 scale)
  • Total Users: Count of all platform users (1.2k format)

Clicking metric cards navigates to filtered views.

2. Live Network Infrastructure

Special card featuring integration with Aruba Network POC:

  • LIVE DATA badge: Indicates real-time production data
  • Activity icon: Visual indicator of live monitoring
  • Description: Explains connection to Aruba network devices and controllers
  • External Link: Opens Aruba Network POC in new tab with security protections (rel="noopener noreferrer")

Security: External link prevents reverse-tabnabbing and clearly indicates it opens outside the platform.

3. Property Status Grid

Visual grid showing all properties with real-time status indicators:

  • Healthy: 0 active incidents (green)
  • Degraded: 1-2 active incidents (yellow)
  • Critical: 3+ incidents or any critical-priority incident (red)
  • Offline: Connectivity loss detected (gray)

Each card shows:

  • Property name and location
  • Status badge
  • Active incident count
  • Last updated timestamp

Clicking a property card navigates to /admin/incidents?propertyId={id}

4. Report Incident

Admins can create incidents for any property with full field access including:

  • All categories (Network, SA-specific, Operational)
  • Property selection across entire portfolio
  • Technician assignment
  • Extended metadata fields

Sidebar Navigation

Overview Section

  • Portfolio Dashboard: Main landing page

Management Section

  • Users & Roles: User management, role assignments, property permissions
  • System Config: Platform configuration settings
  • Integrations: Third-party integrations and API connections

Reporting Section

  • Regional Analytics: Cross-property performance analytics
  • Reports: Generate and export reports
  • Audit Logs: System-wide activity audit trail

Admin Privileges

Admin users have full platform access:

  • View and manage all properties
  • Create, edit, delete users and roles
  • Access all incidents across all properties
  • Configure system settings
  • View audit logs and system analytics
  • Manage integrations and external connections

White-Label Considerations

The platform is designed for white-label deployment:

  • Property branding can be customized per deployment
  • Color schemes defined via CSS variables
  • South African localization (currency, dates, spelling)
  • Multi-property support with property-specific configurations
